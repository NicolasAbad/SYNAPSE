# SYNAPSE

A mobile idle/incremental game where you evolve a neural network from a single neuron to cosmic superintelligence.

**Built 100% with AI assistance** as an experiment in AI-assisted solo game development.

- **Platforms:** iOS, Android (Capacitor)
- **Genre:** Idle / Incremental
- **Target audience:** fans of Cell to Singularity, Antimatter Dimensions, Realm Grinder
- **F2P with cosmetics + convenience IAP:** 100% of content accessible without spending money

## Features

- 3 currencies (Thoughts, Memories, Sparks)
- 5 neuron types with connection multipliers
- 35 upgrades + 4 Run-exclusive + 8 Resonance
- 3 Archetypes (Analítica / Empática / Creativa)
- 15 Mutations × 3 Neural Pathways per cycle
- 3 Runs × 26 prestiges = 78 total prestiges
- 4 endings in v1.0 (3 archetype + 1 Secret; 5th ending "The Witness" in v1.5)
- 57 narrative fragments, 30 Echoes
- 5 Mental States, 8 Micro-challenges
- Weekly Challenges, Daily Login, Personal Bests
- Offline progression with Lucid Dreams
