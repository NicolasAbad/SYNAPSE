Inconsistencias corregidas en el GDD

FIX-QA1: Soft cap formula corregida en 3 lugares

Antes: softCap(x) = 100 + sqrt(x-100) × 20 — amplificaba valores entre 100-500 (bug). Ahora: softCap(x) = 100 × pow(x/100, 0.72) — siempre retorna ≤ x. Corregido en: Constants, PROD-5 rule, y referencia en production code.

FIX-QA2: Production formula usaba Math.min en vez de softCap

Antes: totalMult = Math.min(rawMult, productionCap). Ahora: totalMult = softCap(rawMult). El campo productionCap en constants está marcado DEPRECATED.

FIX-QA3: PROD-1 rule actualizada

Antes: "production = baseRate × min(allMults, cap)". Ahora: "production = baseRate × softCap(allMults). NUNCA Math.min."

FIX-QA4: runThresholdMult extendido para post-launch

Antes: [1.0, 3.5, 6.0]. Ahora: [1.0, 3.5, 6.0, 8.5, 12.0, 15.0] para Runs 1-5+ e Infinito.

FIX-QA5: Patrón Resonante 3 — condición ajustada

Antes: "Solo Básicas" (demasiado difícil, ciclo de 2+ horas). Ahora: "80%+ de las neuronas compradas son Básicas" — todavía difícil pero completable en ~40 minutos. Condición: neurons[0].owned / totalNeuronsOwned >= 0.80.

FIX-QA6: Sinapsis Espontánea "Eureka" — edge case con Funciones Ejecutivas

Si "Eureka" (siguiente upgrade cuesta 0) se activa mientras Funciones Ejecutivas está comprado (−12% costo), el descuento se aplica DESPUÉS de Eureka: 0 × 0.88 = 0. Sin problema. Pero hay que asegurarse de que el código aplique Eureka como un flag (nextUpgradeFree: true), no como un multiplicador de 0×.

FIX-QA7: Focus Bar en prestige — campos faltantes en PRESTIGE_RESET

Agregados a PRESTIGE_RESET: insightEndTime, lastSpontaneousCheck, spontaneousMemoryUsed, spontaneousInterferenceUsed. Sin esto, los flags de "max 1 por ciclo" de Sinapsis Espontáneas no se resetearían.

FIX-QA8: Pattern Decision — UI faltante

Cuando el jugador completa un prestige y el siguiente nodo de Patrón es un nodo de Decisión, se muestra un modal DESPUÉS del Árbol de Patrones y ANTES de Polaridad. "Elige tu camino: [Opción A] o [Opción B]". La decisión es permanente — se muestra un warning "Esta elección es permanente." El modal es skippable solo si el jugador ya decidió en todos los nodos disponibles.

Pantalla por pantalla — especificación para Claude Code

1. Canvas principal (tab Mente)

LAYOUT

Canvas 2D ocupa todo el espacio disponible entre el HUD (top) y la nav bar (bottom). Background: negro puro (#03050c). Nodos: círculos con glow radial. Conexiones: líneas entre nodos de tipos diferentes. Tema evoluciona: bioluminiscente (Era 1) → digital/circuito (Era 2) → cósmico (Era 3).

HUD overlay (sobre el canvas):

TOP-LEFTThoughts counter. Font: JetBrains Mono, 20px, bold, color dorado (#F0A030). Debajo: "pensamientos" en 9px gris.
TOP-RIGHTRate display. Font: mono 13px, verde (#22B07A). "+[rate]/seg".
TOP-CENTERCargas de Disparo: 3-5 círculos (8px) llenos (dorado) o vacíos. Debajo: countdown "19:42" en mono 9px dorado 50%.
BELOW CHARGESFocus Bar (P4+): barra horizontal, 60% del ancho, altura 4px. Color: cyan (#40D0D0). Background: #ffffff08. Al llegar a 100%: pulsa. Si hay niveles 2-3 (P10+): la barra tiene marcas visuales a 100%, 200%.
BELOW FOCUSSinapsis Espontánea activa (si hay): chip con nombre + countdown. Font: 10px, color del tipo (verde/naranja/rojo). Desaparece al terminar.
RIGHT EDGEBarra de consciencia: vertical, semi-transparente, 3px ancho, aparece al 50% threshold. Color: púrpura (#8B7FE8). Al 100%: pulsa dorado.
BOTTOMNav bar: 4 tabs. Mente (siempre) · Neuronas (desde 10 thoughts) · Mejoras (desde 80) · Regiones (desde 1% threshold).

2. Tab Neuronas

Lista vertical de tipos de neurona. Cada tipo muestra: ícono (por tipo), nombre, owned count, rate/s, costo del siguiente. Botón "Comprar" (o "Comprar 10" / "Comprar Max" con toggle). Tipos bloqueados: mostrar silhouette + requisito ("Requiere 5 Básicas"). Solo 1 teaser visible del siguiente tipo bloqueado. Auto-buy toggle (P5+): ícono de cerebro con ⟳ en el header del panel.

3. Tab Mejoras

Lista vertical ordenada: NUEVOS (borde de color, badge "nuevo", últimos 3 min) → COMPRABLES (precio en verde) → DISPONIBLES (precio en naranja, greyed) → COMPRADOS (colapsados, ✓). Cada upgrade: nombre, descripción del efecto, costo, badge de tier (P0/P2/P4/P6/P10). Con Funciones Ejecutivas: upgrades de thoughts muestran precio tachado + precio nuevo en verde.

4. Tab Regiones

3 secciones en scroll vertical: (1) Mapa cerebral simplificado — regiones activas con color, bloqueadas con "???". (2) Historia — fragmentos narrativos leídos (scrollable). (3) Despertares (Memoria Colectiva) — botón "Ver historial" que expande la lista de prestige log. Síntesis de Memoria al final: "Sintetizar (8 Mem → 1 nodo)" si hay nodos disponibles.

5. Modales pre-ciclo (post-prestige)

Secuencia después de hacer prestige:

1Animación de Despertar (3s). Canvas se "reinicia" visualmente.
2Árbol de Patrones. Gastar patrones en nodos. Si hay nodo de Decisión: modal con 2 opciones side-by-side + warning "permanente".
3Polaridad (P3+). Modal: 2 columnas. Excitatorio (dorado) vs Inhibitorio (púrpura). Stats de cada uno. Botón "Usar anterior" (default).
4Mutación (P7+). 3 cartas con efecto y tradeoff. Swipe para leer detalles. Botón "Usar anterior".
5Pathway (P10+). 3 rutas como caminos ramificados. Muestra qué upgrades habilita/bloquea. Botón "Usar anterior".
6Canvas nuevo ciclo. Gameplay empieza.

Si el jugador tapea "Usar anterior" en todos los modales, la secuencia entera toma <3 segundos después de la animación.

6. Pantalla de Despertar (pre-prestige)

Cuando la barra de consciencia llega al 100%: banner inferior "DESPERTAR disponible". Tap → pantalla de confirmación:

Verde+3 Patrones, +2 Memorias, mecánica nueva desbloqueada (si aplica)
Rojo suaveNeuronas y thoughts se resetean
AzulTiempo estimado del próximo ciclo: "~18 min (vs 24 min este ciclo)"
Botón "DESPERTAR" (prominente) + "Más tarde" (pequeño)

Chequeo de Patrones Resonantes: al confirmar Despertar, ANTES de resetear, evaluar las 4 condiciones. Si alguna se cumple → animación dorada + notificación.

7. Pantalla de regreso (Sueño)

Si offline ≥ 60 segundos al abrir la app:

Animación (4s)Canvas muestra el cerebro "soñando" — neuronas parpadean suave, partículas doradas de thoughts caen y se absorben. Counter sube visualmente.
Resumen"Tu mente generó X pensamientos. Y cargas acumuladas." Si Resonancia Acumulada: "Primer Disparo potenciado +Z%."
Sueño Lúcido (P10+, 33%)Elección: "+10% prod por 1h" vs "+2 Memorias".
Ad opcionalSi offline ≥ 30 min: "Ver anuncio para ×2 ganancias." Banner sutil, no popup.
