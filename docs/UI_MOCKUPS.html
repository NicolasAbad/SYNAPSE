SYNAPSE — UI Screen Mockups

6 main screens. SVG wireframes with exact positions for Claude Code.

1. Canvas Principal + HUD

Canvas + HUD. Thoughts (TL), rate (TR), charges+countdown (TC), Focus Bar, Spontaneous Synapse banner, consciousness bar (right vertical), Discharge button, 4 tabs.

2. Awakening Screen

Confirmación. Verde=ganancias, rojo=resets, azul=estimado, naranja=récord, gold dashed=Resonant Pattern (condicional).

3. CycleSetupScreen (unificada P10+)

3 columns (1 at P3, 2 at P7, 3 at P10+). "SAME AS BEFORE" = 1 tap. [change] opens bottom sheet. Insula recommendation (v2.0, dashed). Momentum bonus visible.

4. Regreso Offline

Offline return. Animated counter. Sueño Lúcido (P10+, 33%). Optional ad (subtle). Genius Pass removes ad.

5. Pattern Tree + Decisión

Árbol + Decisión. Nodos normales (verde), hitos (★ dorado), Decisión (naranja). Modal: 2 opciones + warning permanente.

6. Tab Neuronas

Type list. Color-coded. Owned×rate=total. Green button=affordable, orange=expensive, red=can.t afford. Locked=dashed+requirement. Auto-buy toggle (P5+). ×1/×10/Max.

For Claude Code: Canvas uses Canvas 2D (not SVG). Panels are React+scroll. Modals with backdrop blur. Haptics on every interaction. Numbers with formatNumber() (K/M/B/T). Exact colors in CSS vars from GDD.
