Final Main HUD

Tab content redesigns (4 tabs)

Each tab opens at top:18% covering ~75-82% of viewport. Content patterns adapted from the original UI mockups + 2026 idle-game best practices (compact rows, scan-friendly grouping, color-coded affordability).

Neurons tab — compact + best-buy

Mode selector at top (×1/×10/MAX). Compact rows. ★ best-DPS-per-thought badge. Locked rows dim with explicit unlock hint.

Upgrades tab — affordable count

Affordable count badge in header. Affordable section first. "Coming next" teaser dimmer. Locked hint at bottom.

Mind tab — rooms listNEW

Replaces horizontal subtab pills with a vertical rooms list. Each room → drill into full sub-page. Locked rooms grouped at bottom. NGU Idle pattern.

Regions tab — brain mapNEW

Brain-map canvas overview replaces vertical card list. Owned regions illuminated. Tap a node → drill into region detail page. Cross-region meta upgrade card below.
