Mi A3 narrow-screen target (720×1520, mockups shown at 360×720 half-resolution)

Tutorial flow — "The Mind Awakens"

Three pivotal phases of the proposed 8-phase tutorial. Each UI element appears as the player's mind "remembers" what it can do — driven by the player's own actions, never by timed prompts.

Phase 0 — Pre-thought

Cold start. Pure black canvas + one faint cyan neuron + italic narrative line. No HUD chrome at all. Time budget: 3-8s before first tap.

Phase 3 — Buy first neuron

Pivotal moment. Player has 10 thoughts (counter shown small top-left, "thoughts" label appeared in Phase 2). Big +1 neuron button overlay invites the first purchase. Single-tap goal.

Phase 7 — First Discharge

Climax. Focus bar at 100%, "Discharge ready" chip pulses, big DISCHARGE button with the SUPERCHARGED ×3 tutorial badge. Bottom tabs visible (Neurons + Upgrades earned in earlier phases).

Final UI — post-tutorial launch state

What the player sees from the first Awakening onward. Hero pair (thoughts + rate) on top, thin amber awakening progress bar, status row of 3 chips, focus bar, discharge chip, canvas with auto-scaled spiral, discharge button only when ready, 4-tab bottom bar.

Final Main HUD

The polished launch HUD. All elements respect narrow-screen layout (no overlap), one-handed reach (action button bottom-center), settings top-right (mobile convention).

Tab content redesigns

Mockups for the two most-touched tabs. Mind tab restructure (rooms-list pattern) and Regions brain-map are described in the proposal but not yet drawn — generate on request.

Neurons tab — compact + best-buy

Mode selector at top (×1 / ×10 / MAX). Each row 3 lines instead of 5. ★ badge marks best DPS-per-thought now. Locked neurons dim with explicit unlock requirement. Sticky effective-rate footer.

Upgrades tab — affordable count + sections

Affordable count badge in header (3 affordable). Affordable section first, scannable in one look. "Coming next" teaser section dimmer. Locked-section hint at bottom.
