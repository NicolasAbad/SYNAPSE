## Design language used (from GDD §3)

| Token | Value | Purpose |
|---|---|---|
| Background deep | `#0A0E1A` | Night canvas |
| Background mid | `#121C36` | Gradient center |
| Sky blue accent | `#4A9FFF` | Primary text, brand |
| Glow | `#7BD4FF` | Particles, soft glow |
| Text dim | `#AAC3E6` | Body copy |
| Text muted | `#6E8CB4` | Disclaimers, captions |

Aesthetic: **bioluminescent** — dark background with soft blue accents, scattered
particle effects evoking synapses firing in a neural network at night.

## What counts as "acceptable final"

- **Icon:** Apple Review prefers simple, legible at 60×60 home screen. The current placeholder is 1 step above minimum; a Fiverr designer can produce a proper animated neuron silhouette version for ~$100.
- **Screenshots:** Apple and Google require real in-game imagery in 2-3 of the 5 screenshots (their policies). Text-only screenshots like these placeholders are acceptable for 1-2 slots (usually the first as a "hero" and the last as a CTA), but the middle 2-3 need actual gameplay.
